## 3. X870E 芯片组强制项（全系一致，不构成差异）

以下几项是 AMD 对 X870E 的**硬性规定**，13 款板子完全一样，**不要拿它们当选板依据**：

- **USB4 40Gbps × 2**（全系标配）
- **CPU 直连 PCIe 5.0 x16 显卡槽**
- **至少 1 条 CPU 直连 PCIe 5.0 x4 M.2**
- **SATA 统一 4 口**（6Gb/s；想要 6 口以上 SATA 的，X870E 全系都满足不了）
- **双 Promontory 21 芯片组** → 全系最小 ATX
- **CPU 供电接口全系 8Pin × 2（EPS 8+8）**——13 款均为 `2 x 8-pin +12V`，唯 APEX 额外多一个 `1 x 8-pin PCIe Power` 辅助供电接口
- **内存全系 4 DIMM / 最大 256GB DDR5**——唯 APEX 例外（2 DIMM / 128GB）

## 5. 三个真正的分水岭

在 SATA 数量、USB4、第一条 PCIe 5.0 x16 全系相同的前提下，**只有下面 3 项真正区分档次**：

### 分水岭 1：第二条 x16 是否 CPU 直连 PCIe 5.0
- **是真 x8/x8 的**：ROG Crosshair 全系 + **ProArt X870E-Creator**
- **不是**：Strix / TUF 的第二条 x16 只是**物理 x16、电气 PCIe 4.0 x4，且走芯片组**
- 影响对象：双显卡、PCIe 5.0 扩展卡（高速采集卡、网卡、M.2 扩展卡）用户

### 分水岭 2：CPU 直连 PCIe 5.0 M.2 的数量
- **3 条**：Hero / Hero BTF / Strix X870E-E / Extreme / Apex
- **2 条**：其余全部
- 注意：第 3 条 CPU M.2 是**从显卡的 16 条通道里借的**（见下节）

### 分水岭 3：万兆（10G）有线网口
- **有**：GLACIAL（双 10G）、EDITION 20（双 10G）、EXTREME（10G+5G）、DARK HERO（10G+5G）、ProArt（10G+2.5G）
- **没有**：其余全部（5G / 2.5G）
- 现实提醒：家里没有 10G 交换机 / NAS 的话，这一项等于白花钱

## 6. 9900X + RTX 5090 专项建议（重点）

### 6.1 平台事实

**AMD Ryzen 9 9900X**
- Zen 5 架构，12 核 24 线程
- TDP **120W**，PPT 162W
- **无 3D V-Cache**
- AM5 接口，PCIe 5.0
- 总计 **28 条可用 PCIe 通道** = 16（显卡）+ 4（CPU 直连 M.2）+ 4（芯片组链路）+ 4（额外 CPU M.2 / 其他）

### 6.2 五条关键推论

#### ① 供电相数：16+2+1 的 80A 方案已经严重过剩，别为相数付钱

9900X 满载 PPT 只有 162W。以 TUF / Strix X870E-H 的 **16 相 × 80A ≈ 1280A** 理论供电能力计算，实际负载占用率极低，温度余量充足。

Crosshair 系列 20~24 相 110A（2000A+）的供电堆料，是给 **液氮/极限超频、以及 16 核 9950X 长时间 AVX-512 全核满载**准备的。对一颗 120W TDP 的 9900X，**多出来的相数不会带来任何频率、温度或稳定性上的可感知收益**。

> 结论：供电这一项，Strix / TUF 档位已经完全够用，不需要往上加钱。

#### ② RTX 5090 只需要一条 PCIe 5.0 x16 → 第二条 CPU 直连 x16 对你无效

RTX 5090 是单卡，占用第一条 CPU 直连 PCIe 5.0 x16 即可满血运行。

而 **Crosshair 全系和 ProArt 相对 Strix 最贵的那部分溢价，恰恰就是「第二条 CPU 直连 x16，支持 x8/x8 拆分」**。单卡用户永远不会用到它。

> 结论：这条最大卖点在你的配置下价值为 0。除非你未来打算插第二块计算卡 / 高速 AIC。

#### ③ 通道冲突警告：别把 3 条 CPU 直连 M.2 插满

Ryzen 9000 只有 28 条通道。在 Hero / Strix X870E-E / Extreme 这类「3 条 CPU 直连 5.0 M.2」的板子上，**插满第 3 条 CPU M.2 会触发 x8+x4+x4 拆分，显卡直接从 x16 掉到 x8**。

对 RTX 5090：
- PCIe 5.0 x8 的带宽（约 32GB/s 单向）理论上够用
- 实测性能损失通常在 **1–3%**，个别高帧率场景略高

> 建议：**只用 2 条 CPU 直连 M.2（这两条走的是独立的 4 条通道，不动显卡），第 3 块及以后的 SSD 走芯片组 PCIe 4.0 x4 槽**。PCIe 4.0 x4 的 SSD 在游戏和日常场景下与 5.0 的体感差异极小，保住显卡满 x16 更划算。
>
> 各板 lane sharing 规则不同，**装机前务必查该型号说明书的 lane sharing 表**。

#### ④ RTX 5090 是三槽以上厚卡 → 优先看 M.2 散热布局和免工具卸载

5090 公版/非公普遍 3–4 槽厚，实际后果：
- **会完全压住第一条 M.2 散热片**（位于显卡正下方），高负载时 SSD 温度上升明显
- **会挡住第二条 PCIe 槽**，甚至连第三条也够不着（对单卡无所谓，但影响声卡/采集卡）
- 拆装显卡时，PCIe 卡扣被显卡散热鳍片挡住，非常难按

选板要点：
1. 第一条 M.2 是否有**独立厚散热马甲**，最好带背面散热（Dark Hero 的 3D VC 均热板是这一档最好的）
2. 是否有 **Q-Release / Q-Release Slim 免工具显卡卸载**——对 5090 这种重卡是刚需体验
3. 优先把主力 SSD 装在**不被显卡覆盖**的 M.2 位

#### ⑤ 电源与供电接口

- RTX 5090 TGP **575W**，+ 9900X 满载约 160W + 其余
- **整机建议 1000W 以上电源**（推荐 1000–1200W ATX 3.1 / 原生 12V-2×6 接口）
- 主板端需确认 **CPU 供电为双 8Pin（EPS 8+8）**——本文列出的 Strix / TUF / ProArt / Crosshair 全系均满足

### 6.3 最终推荐

> **首选：ROG Strix X870E-H GAMING WIFI7（¥2,949）**

理由：
- 16+2+1 80A 供电对 120W 的 9900X 绰绰有余
- 2 条 CPU 直连 PCIe 5.0 M.2 + 2 条芯片组 4.0 M.2 = 4 槽，且**天然不会跟显卡抢通道**，5090 稳定满 x16
- 后置 USB 14 个（这一档最多）、WiFi7、Realtek 5G 网口
- X870E 芯片组强制的 USB4 ×2 一个不少
- 省下来的 ¥1,000~7,000 直接加到 SSD / 内存 / 电源上，收益远高于主板堆料

如果喜欢初音未来联名外观，同规格版本 ¥3,199，贵 ¥250 纯买外观。

> **平价备选：ROG Strix X870E-A GAMING WIFI7 NEO（¥2,999）**

规格与 -H 基本同级（供电略强 16+2+2，内存超频标称 8600+ 更高，WiFi 模块 MT7925 略低一档）。白色配色，配白色机箱更协调。两者选哪个基本看外观和当时实际到手价。

> **升级备选：ROG Strix X870E-E GAMING WIFI（¥3,899）**

**只有一个理由值得多花这 ¥900**：你需要 **3 条 PCIe 5.0 SSD**（例如大量本地 AI 模型权重、8K 视频素材盘）。
但要记住第 ③ 条警告——插满第 3 条就意味着显卡降到 x8。更理性的做法仍然是买 X870E-H，把预算给 SSD 本身。

> **特定需求才上：ROG Crosshair X870E DARK HERO（¥4,899）/ ProArt X870E-Creator（¥4,699）**

- 选 **Dark Hero**：家里已有 **10G 网络**，或者对 M.2 散热（3D VC 均热板）和 ROG 顶级用料/BIOS 有偏执要求
- 选 **ProArt**：需要 **真 x8/x8 双卡**（第二块计算卡）或高速 AIC，且要 10G 网口。这是唯一非 Crosshair 但具备 CPU 直连双 x16 的板子，性价比明显优于 Crosshair

> **明确不推荐：X870E EXTREME（¥9,999）/ GLACIAL（¥10,499）/ EDITION 20**

对「单卡 5090 + 120W 的 9900X」这个组合，这三块板提供的全部增量（24 相供电、E-ATX 布局、DIMM.2、双 10G、水冷头）**没有一项能转化成性能**。¥7,000 的差价足够把 5090 的电源、SSD、内存全部升到顶级，或者直接换更强的 CPU。这是纯粹的浪费。

**X870E APEX** 同理排除：它是 2 DIMM 内存超频特化板（DDR5-9600+），代价是内存容量上限 128GB 且只能插两条；除非你打算冲内存频率世界纪录，否则没有理由选它。

### 6.4 一句额外的话：CPU 搭配

如果这台机器的**主要用途是游戏**，9900X 配 RTX 5090 是明显偏科的组合——**9800X3D 或 9950X3D 凭借 3D V-Cache，在 1080p/1440p 高刷场景下的游戏帧数会显著高于 9900X**，而 5090 恰恰是最容易被 CPU 拖后腿的卡。

9900X 的定位是**游戏 + 生产力混合负载**（多核渲染、编译、虚拟机）。如果你的实际使用确实是这种混合场景，那这个搭配是合理的；如果纯游戏，建议在下单前重新考虑 CPU。

### 通用注意事项

**Ryzen 9000 系列仅 28 条可用 PCIe 通道**。多款主板的第 3 条 CPU 直连 M.2 会与显卡共享通道（触发 x8+x4+x4 拆分）。本文表格中的槽位数量是**物理槽位**，不代表可以同时满速使用。

**装机前必做**：下载目标型号的官方 User Manual，查阅其中的 **lane sharing / bandwidth allocation 表**，确认你的 SSD + 显卡 + 扩展卡组合不会互相降速或直接禁用某个槽位。
