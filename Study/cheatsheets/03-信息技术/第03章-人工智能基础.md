## 一、本章概览

本章是信息技术专题中人工智能模块的入门章，系统介绍人工智能的基本概念、发展历程、分类与流派，进而深入机器学习的核心方法论（监督/无监督/强化学习、损失函数、过拟合等），最后落脚于神经网络与深度学习的关键模型与训练机制（激活函数、前馈网络、反向传播等）。全章分三节：**第一节 人工智能**（概念·应用·发展·分类·流派·计算方法·专家系统）、**第二节 机器学习**（定义·三要素·分类·损失函数·拟合问题·正则化·生成式与判别式模型）、**第三节 神经网络与深度学习**（ANN·深度学习·MP模型·激活函数·感知机·前馈网络·梯度下降·反向传播·常见模型·ChatGPT）。三节构成"AI 总论 → 机器学习方法论 → 神经网络与深度学习技术"的递进结构。

## 二、知识点详解

### 第一节 人工智能

### 1. 人工智能的概念 ⭐(练习1,2)(判断1)

- **定义**：人工智能是以机器为载体实现的人类智能（或生物智能）。英文缩写为 **AI**（Artificial Intelligence）。它是研究、开发用于模拟、延伸和扩展人的智能的理论、方法、技术及应用系统的一门新技术。
- **要点**：
  - 人工智能是**计算机科学的一个分支**（不是统计学、数学或生物学分支）。
  - 它企图了解智能的实质，并生产出一种新的能以人类智能相似的方式做出反应的智能机器。
  - 该领域的研究包括：**机器人、语言识别、图像识别、自然语言处理和专家系统**等。
  - 人工智能由不同的领域组成，如机器学习、计算机视觉等。
  - 主要目标：使机器能够胜任一些通常需要人类智能才能完成的复杂工作。

### 2. 人工智能的应用领域

- **应用领域**包括：机器翻译、智能控制、专家系统、机器人学、语言和图像处理、遗传编程、机器人工厂、自动程序设计、航天应用、信息处理、储存与管理、执行生命体无法执行的或复杂或规模庞大的任务等。

### 3. 人工智能的发展 ⭐(练习3)

- **1956 年达特茅斯会议**提出人工智能概念。
- 会议主要发起人**约翰·麦卡锡（John McCarthy）**被誉为**人工智能之父**。
- 发展阶段（按时间轴）：
  - **1956**：人工智能诞生
  - **1960s**：初春（起步发展期）
  - **1970s**：初冬/寒冬（机器翻译笑话百出、任务失败目标落空 → 反思发展期/低迷发展期）
  - **1980s**：复苏（专家系统遍地开花 → 应用发展期）
  - **1990s**：寒冬（专家系统发展乏力、神经网络研究受阻 → 低迷发展期）
  - **2000s**：复苏（互联网推动人工智能；IBM 提出智慧地球；我国提出感知中国；物联网、云计算、大数据 → 稳步发展期）
  - **2010s**：爆发（深度学习和大数据兴起带来了人工智能的爆发；不断创新和实用 → 蓬勃发展期）

### 4. 人工智能分类 ⚠(判断2)

- **弱人工智能（ANI）**：也称限制领域人工智能或应用型人工智能。
  - 本质：某个特定领域内基于统计规律的大数据处理。
  - 特征：依葫芦画瓢、任务导向，数据与规则驱动；只专注于完成某个特定的任务。
  - 底层原理：从海量数据中归纳出模型，再泛化至新的数据中进行正向运算。
  - 典型例子：**谷歌的 AlphaGo** 是典型"弱人工智能"——虽能战胜围棋世界冠军，但仅限于单个游戏领域。语音识别、图像识别和翻译也属此类。
- **强人工智能（AGI）**：又称通用人工智能。
  - 特征：举一反三、从经验中学习，学会学习。
  - 人类级别的人工智能，拥有**独立思想和意识**，在各方面均能与人类媲美。
  - 机器本身可拥有"思维"，能够进行独立的思考、计划、解决问题、抽象思维、理解复杂理念、快速学习等，可实现"全面仿人性"。
  - 目前只存在于电影及人类想象中。
- **超人人工智能**：
  - 假设计算机程序通过不断发展，智力水平可以超越人类。
  - 定义最为模糊，目前还没有精准预测，只能从哲学或科幻的角度加以想象。

### 5. 人工智能的三大流派 ⭐(练习4,5)(多选2)(判断7)

- **符号主义**：认为人工智能源于**数学逻辑**。实质是模拟人的抽象逻辑思维，用符号描述人类的认知过程。→ 数学逻辑 → 知识表示
- **连接主义**：认为人工智能源于**仿生学**。模拟人脑神经系统的结构和功能。→ 仿生学 → 神经网络
- **行为主义**：认为人工智能源于**控制论**。智能行为产生的基础是"从感知到行动"的反应机制，智能是在与外界环境交互作用中表现出来的。→ 控制论 → 自主机器人

| 学习模式 | 优势 | 不足 |
|---|---|---|
| 符号主义 | 与人的逻辑推理相似，解释性强 | 难以构建完备的知识规则库 |
| 连接主义 | 直接从数据中学 | 依赖于数据、解释性不强 |
| 行为主义 | 从经验中进行能力的持续学习 | 非穷举式搜索，需要更好的策略 |

> ⚠ **易错**：连接主义源于**仿生学**（不是数学逻辑）；符号主义才源于数学逻辑。(判断7)

### 6. 可计算载体：形式化

- **非形式化**：自然语言表达。
- **半形式化**：具有确定语义定义并有严格语法的语言表达。
- **形式化**：在完备数学概念基础上，具有确定语义定义并有严格语法的语言表达。

### 7. 可计算载体：机械化

- **图灵机**以机械方式进行"计算"，成为现代计算机理论模型，人工智能的"机器载体"。

### 8. 智能计算方法 ⭐(练习6)(多选3)

智能计算方法的五个方向：

| 方向 | 核心 | 说明 |
|---|---|---|
| 逻辑推理 | 符号主义为核心 | 从已知判断（前提）推出新判断（结论）的过程。主要有归纳法和演绎法 |
| 探索搜索 | 问题求解为核心 | 依据已有信息寻找满足条件约束的答案。分为**盲目搜索、启发式搜索、对抗搜索** |
| 机器学习 | 数据驱动为核心 | 从大量数据中发现其所承载语义的内在模式，并利用该模式完成识别或分类任务 |
| 强化学习 | 行为主义为核心 | 根据环境的奖罚反馈，学习可施加的最佳行动，进而形成了自我学习的能力 |
| 决策智能 | 博弈对抗为核心 | 从"优化解"向"均衡解"转变。与博弈论深度融合 |

### 9. 专家系统 ⭐(练习7)

- **定义**：专家系统是早期人工智能的一个重要分支，它可以看作是一类具有专门知识和经验的计算机智能程序系统。
- **技术**：一般采用人工智能中的**知识表示**和**知识推理**技术来模拟通常由领域专家才能解决的复杂问题。

### 10. 人工智能、机器学习与深度学习 ⭐(判断3)

- **机器学习**是一种实现人工智能的方法。
- **深度学习**是一种实现机器学习的技术。
- 三者为**包含关系**：人工智能 ⊃ 机器学习 ⊃ 深度学习。

### 第二节 机器学习

### 11. 机器学习

- **定义**：从数据中自动学习知识的一种人工智能方法。
- 机器学习从数据中提取特征，寻找数据和任务目标之间的关系：**函数 $f$**。

### 12. 机器学习方法三要素 ⭐(练习8)

- **模型**：用来描述客观世界的数学模型。
- **策略**：损失函数和正则化项。
- **算法**：最优化算法。

> ⚠ **易错**：三要素是模型、策略、算法，**不包括"数据"**。(练习8)

### 13. 机器学习分类 ⭐(多选5)

- **有监督学习**（Supervised Learning）：有标签
- **无监督学习**（Unsupervised Learning）：无标签
- **半监督学习**（Semi-supervised Learning）
- **强化学习**（Reinforcement Learning）

### 14. 监督学习 ⭐(练习9)(多选7)

- **功能分类**：**分类**（Classification）、**回归**（Regression）
- **常见方法**：线性回归、逻辑回归、**决策树**、线性判别分析、AdaBoost、支持向量机。

### 15. 无监督学习

- **功能分类**：**聚类**（Clustering）、**降维**（Dimensionality）
- **常见方法**：K 均值聚类、主成分分析、期望最大化算法。

### 16. 强化学习 ⭐(多选6)(判断6)

- **定义**：又称**再励学习、评价学习或增强学习**，是机器学习的范式和方法论之一。
- 用于描述和解决智能体（agent）在与环境的交互过程中通过学习策略以达成回报最大化或实现特定目标的问题。

### 17. 损失函数（Loss Function）⭐(练习10)(判断4)

- 假设 $y_l = f(x_l)$。
- 损失函数用来计算**真实值 $y_l$** 与**预测值 $f(x_l)$** 间的差值。

### 18. 训练集、测试集 ⭐(练习11)

- **训练集**：用于模型训练。
- **测试集**：用于模型泛化能力评估。

### 19. 过拟合、欠拟合 ⭐(练习12)(判断5)

| 训练集上的表现 | 测试集上的表现 | 结论 |
|---|---|---|
| 不好 | 不好 | **欠拟合** |
| 好 | 不好 | **过拟合** |

> ⚠ **易错**：过拟合是训练集好、测试集**不好**（不是都好）。(判断5)

### 20. 正则化（Regularization）⭐(练习13)

- **定义**：通过限制模型**复杂度**，从而避免过拟合、提高泛化能力的方法。

### 21. 生成式模型、判别式模型

- **生成式模型**（generative model）：通过对观测值和标注数据计算**联合概率分布 $P(x,y)$** 估算 $y$。
- **判别式模型**（discriminative model）：通过求解**条件概率分布 $P(y|x)$** 或者直接计算 $y$ 的值预测 $y$。

### 第三节 神经网络与深度学习

### 22. 神经网络 ⭐(多选8)

- **定义**：人工神经网络（ANN）也简称为**神经网络**或称作**连接模型**，它是一种模仿动物神经网络行为特征，进行**分布式并行信息处理**的算法模型。
- 这种网络依靠系统的复杂程度，通过调整内部大量节点之间相互连接的关系，达到处理信息的目的。
- 人工神经网络是生物神经网络在某种简化意义下的技术复现。
  - **生物神经网络**主要研究智能的**机理**。
  - **人工神经网络**主要研究智能机理的**实现**。
  - 两者相辅相成。

### 23. 深度学习

- 深度学习是机器学习领域中一个新的研究方向，使机器学习更接近于最初的目标：人工智能。
- 深度学习是学习样本数据的内在规律和表示层次，这些学习过程中获得的信息对诸如文字、图像和声音等数据的解释有很大的帮助。
- 最终目标：让机器能够像人一样具有分析学习能力，能够识别文字、图像和声音等数据。
- 深度学习是一个复杂的机器学习算法，在**语音和图像识别**方面取得的效果，远远超过先前相关技术。
- 应用领域：搜索技术、数据挖掘、机器学习、机器翻译、自然语言处理、多媒体学习、语音、推荐和个性化技术等。

### 24. 神经网络与深度学习的关系 ⭐(练习16)(判断8)

- **神经网络**：一种以（人工）神经元为基本单元的模型。
- **深度学习**：一类机器学习问题，主要解决**贡献度分配问题**。

### 25. 深度学习发展史 ⭐(练习15,17)

| 年份 | 事件 |
|---|---|
| 1940 年 | 提出神经元的结构，但权重是**不可学**的 |
| 1950 年 | 提出权重学习理论，神经网络结构趋于完善，开启神经网络**第一个黄金时代** |
| 1969 年 | 提出**异或问题**，神经网络模型进入**黑暗时代** |
| 1986 年 | 多层神经网络解决异或问题 |
| 1990 年 | 支持向量机兴起，神经网络未得到重视 |
| 2010 年 | 深度学习兴起。神经网络模型重新焕发生机 |

### 26. 神经元、MP 模型 ⭐(练习18)

- 神经科学家 **McCulloch** 和逻辑学家 **Pitts** 提出 **"McCulloch-Pitts neuron"** 模型，简称 **MP 模型**。
- 生物神经元与数学模型的对应关系：
  - **树突（dendrite）** → 输入 $x_i$
  - **突触（synapse）** → 权重 $w_i$
  - **细胞体（cell body）** → 求和 $\sum w_i x_i + b$
  - **激活函数（activation function）** → 非线性变换
  - **轴突（axon）** → 输出

### 27. 激活函数 ⭐(多选9)(判断9)

- **定义**：对输入信息进行**非线性变换**。

> ⚠ **易错**：激活函数进行的是**非线性**变换，不是线性变换。(判断9)

| 名称 | 函数 | 导数 | 值域 |
|---|---|---|---|
| Sigmoid | $f(x)=\dfrac{1}{1+e^{-x}}$ | $f'(x)=f(x)\cdot(1-f(x))$ | $(0,1)$ |
| Tanh | $f(x)=\dfrac{1-e^{-2x}}{1+e^{-2x}}$ | $f'(x)=1-f(x)^2$ | $(-1,1)$ |
| ReLU | $f(x)=\begin{cases} 0, & x \le 0 \\ x, & x > 0 \end{cases}$ | $f'(x)=\begin{cases} 0, & x \le 0 \\ 1, & x > 0 \end{cases}$ [原文 OCR 疑似遗漏导数符号撇号] | $[0,+\infty)$ |

### 28. 单层感知机

- 结构：输入 $x_1, x_2, \ldots, x_d$ → 权重 $w_1, w_2, \ldots, w_d$ → 求和 $\sum$ → 激活函数 $f$ → 输出 $a$。

### 29. 多层感知机 ⭐(多选10)(判断10)

- **结构**：**输入层、输出层、至少一个隐藏层**。
- 如果没有隐藏层，就成了单层感知机。

> ⚠ **易错**：多层感知机**至少要有一个隐藏层**。(判断10)

### 30. 传统的浅层学习与深度学习

| | 传统浅层学习 | 深度学习 |
|---|---|---|
| 学习方式 | 分段学习 | 端到端学习 |
| 特征提取 | **人工**提取特征 | **自动**提取特征 |
| 流程 | Input → Feature extraction → Classification → Output | Input → Feature extraction + Classification → Output |

### 31. 前馈神经网络 ⭐(练习19)

- 也称**全连接神经网络**、**多层感知器**。
- **结构特征**：
  - 各神经元分别属于不同的层。
  - **层内无连接**。
  - 相邻两层之间的神经元**全部两两连接**。
  - 整个网络中**无反馈**。
  - 信号从输入层向输出层**单向传播**。

### 32. 梯度下降

- **核心**：梯度反方向 = 误差下降最快。

### 33. 反向传播 ⭐(练习20)

- **反向传播**（Backpropagation，BP）是"**误差反向传播**"的简称。
- 与最优化方法（如**梯度下降法**）结合使用。
- 用来训练人工神经网络的常见方法。
- 该方法对所有权重计算损失函数的梯度，这个梯度会反馈给最优化方法，用来更新权值以最小化损失函数。

### 34. 深度学习的参数优化与学习

- 通过**损失函数**算误差，从输出端出发：利用**反向传播**将误差传递给神经网络中的每个单元，使用**梯度下降**更新参数。

### 35. 学习率

- 学习率 $\eta$ 的影响（据原文图表）：
  - $\eta$ **过小**：收敛缓慢
  - $\eta$ **适中**：理想收敛
  - $\eta$ **较大**：可能震荡
  - $\eta$ **过大**：可能发散/越过最优解

### 36. 常见的深度学习模型

| 缩写 | 全称 |
|---|---|
| CNN | 卷积神经网络 |
| RNN | 循环神经网络 |
| LSTM | 长短时记忆网络 |
| GRU | 门控循环单元 |
| GAN | 生成对抗网络 |

### 37. ChatGPT

- **GPT**：生成式预训练 Transformer（Generative Pretrained Transformer）。
- **ChatGPT** 是美国人工智能研究实验室 **OpenAI** 新推出的一种人工智能技术驱动的自然语言处理工具。
- ChatGPT 不单是聊天机器人，还能进行撰写邮件、视频脚本、文案、翻译、代码等任务。

| 判别式模型 | 条件概率分布 $P(y\|x)$ 或直接计算 $y$ 预测 $y$ | 知识点11（第二节） |
| Sigmoid 函数 | $f(x)=\dfrac{1}{1+e^{-x}}$，导数 $f'(x)=f(x)\cdot(1-f(x))$，值域 $(0,1)$ | 知识点6（第三节） |
| Tanh 函数 | $f(x)=\dfrac{1-e^{-2x}}{1+e^{-2x}}$，导数 $f'(x)=1-f(x)^2$，值域 $(-1,1)$ | 知识点6（第三节） |
| ReLU 函数 | $f(x)=\begin{cases} 0, & x \le 0 \\ x, & x > 0 \end{cases}$，值域 $[0,+\infty)$ | 知识点6（第三节） |
| MP 模型 | 输出 = 激活函数$(\sum w_i x_i + b)$ | 知识点5（第三节） |
| 梯度下降 | 梯度反方向 = 误差下降最快 | 知识点11（第三节） |
| 反向传播（BP） | 误差反向传播，与梯度下降法结合使用，对所有权重计算损失函数的梯度，更新权值以最小化损失函数 | 知识点12（第三节） |
| 深度学习参数优化流程 | 损失函数算误差 → 反向传播传递误差 → 梯度下降更新参数 | 知识点13（第三节） |

## 六、一句话记忆要点

1. **AI 概念**：AI = Artificial Intelligence，计算机科学分支，1956 年达特茅斯会议诞生，麦卡锡为父。
2. **AI 分类**：弱 AI（ANI）专一任务、强 AI（AGI）人类级别有意识、超人 AI 超越人类——AlphaGo 是弱 AI。
3. **三大流派**：符号主义=数学逻辑→知识表示；连接主义=仿生学→神经网络；行为主义=控制论→自主机器人。
4. **智能计算五方向**：推理（符号）、搜索（问题求解）、机器学习（数据）、强化学习（行为）、决策智能（博弈）。
5. **AI/ML/DL 关系**：AI ⊃ ML ⊃ DL，层层包含。
6. **ML 三要素**：模型 + 策略（损失函数+正则化项）+ 算法（最优化）——不含数据。
7. **监督 vs 无监督**：有标签→分类/回归；无标签→聚类/降维。
8. **过拟合 vs 欠拟合**：都不好=欠拟合；训练好测试差=过拟合；正则化限制复杂度来防过拟合。
9. **生成式 vs 判别式**：生成式用 $P(x,y)$，判别式用 $P(y|x)$。
10. **激活函数**：Sigmoid $(0,1)$、Tanh $(-1,1)$、ReLU $[0,+\infty)$——非线性变换。
11. **前馈神经网络**：全连接、层内无连接、层间两两连接、无反馈、单向传播。
12. **训练核心流程**：损失函数算误差 → 反向传播传误差 → 梯度下降更新参数。
13. **深度学习关键**：端到端学习、自动提取特征、解决贡献度分配问题。
14. **深度学习模型**：CNN（卷积）、RNN（循环）、LSTM（长短时记忆）、GRU（门控循环单元）、GAN（生成对抗）。
15. **ChatGPT**：GPT = Generative Pretrained Transformer，OpenAI 出品。
