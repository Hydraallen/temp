
三、判断题
1、对称密钥体制是加密密钥与解密密钥相同的密码体制，其中密钥和算法都是保密的。( )
2、公钥密码体制中，加密密钥是公开的，解密密钥是保密的。( )
3、数字签名必须保证能够实现报文鉴别、报文的完整性和不可否认。( )
4、认证中心CA将私钥与其对应的实体(人或机器)进行绑定。每个实体都有CA发来的证书，里面有公钥及其拥有者的标识信息(人名或IP地址)。( )
5、IPsec是网络层安全协议，支持IPv4和IPv6。( )
6、SSL广泛应用于基于万维网的各种网络应用，作用在端系统应用层的HTTP和运输层之间。( )
7、代理服务器可以实现基于应用层数据的过滤和高层用户鉴别。( )
8、对称密钥可以实现多对一的单向保密通信。( )
9、入侵检测系统IDS是在入侵已经开始，但还没有造成危害或在造成更大危害前，及时检测到入侵，以便尽快阻止入侵，把危害降低到最小。( )
10、PGP是一个完整的电子邮件安全软件包，包括加密、鉴别、电子签名和压缩等技术。( )
