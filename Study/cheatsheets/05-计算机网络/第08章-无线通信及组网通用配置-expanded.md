## 一、本章概览

本章覆盖局域网基础（概念、拓扑、分类、IEEE 802 标准体系）、以太网介质访问控制（CSMA/CD 与令牌环）、四种经典以太网组网方式及快速以太网、无线局域网（WLAN 概念与 802.11 协议族）、以及三种广域网接入技术（ISDN、ADSL、HFC）。内容以概念辨析与标准记忆为主，CSMA/CD 与 CSMA/CA 的区别、令牌环负载特性、IEEE 802 标准对应关系是高频考点。

## 二、局域网基础

**局域网概念**：在小范围内将多种通信设备互连起来，实现数据通信和资源共享的计算机网络。*(讲义 p141)*

**局域网特点**：覆盖范围小；只能够提供物理层、数据链路层各网络层的通信功能；可以连入多种数据通信设备；数据传输速率高，误码率较低可靠性高；协议简单、结构灵活、建网成本低、周期短、便于管理和扩充。*(讲义 p141)*

**局域网拓扑结构**：局域网常采用广播型的拓扑结构，常见的有总线型、星型和环型拓扑结构 3 种。*(讲义 p141)*

**局域网分类方式**：*(讲义 p141)*

| 分类依据 | 类型 | 说明 |
|---|---|---|
| 介质访问控制方法 | 以太网 | CSMA/CD 介质访问控制方法 |
| 介质访问控制方法 | 令牌环网 | 通过"令牌"让各结点获得数据发送权，避免冲突 |
| 网络资源管理方式 | 对等式 | 所有结点地位平等，任何结点之间可直接通信和资源共享，各结点拥有自主权 |
| 网络资源管理方式 | 非对等式 | 结点地位不同（如服务器与客户机），服务器集中控制管理网络资源；"客户机/服务器"网络是典型代表 |
| 网络传输技术 | 基带局域网 | 采用数字信号基带传输技术 |
| 网络传输技术 | 宽带局域网 | 采用模拟信号频带信号传输技术 |

> Caveat: 非对等式网络管理分散，任何人都可以成为管理者，需要授权；资源相对集中；多见办公室、家庭。*(讲义 p141)*

**IEEE 802 标准**：描述了局域网参考模型，已被 ISO 接纳为国际标准，称为 ISO 802 标准。*(讲义 p141-142)*

| 工作组 | 研究内容 |
|---|---|
| 802.1 | 高层接口(HILI)，包括网络结构、网际合作和 LAN 的网络管理 |
| 802.2 | 逻辑链路控制(LLC) |
| 802.3 | 载波监听多路访问/冲突检测(CSMA/CD) |
| 802.4 | 令牌总线 |
| 802.5 | 令牌环 |
| 802.6 | 城域网(MAN) |
| 802.7 | 广域技术建议组(BBTAG) |
| 802.8 | 光纤技术建议组(FOTAG) |
| 802.9 | 综合业务局域网接口(ISIAN) |
| 802.10 | 交互性局域网安全性标准(SILS) |
| 802.11 | 无线局域网(WLAN) |
| 802.12 | 命令优先级 |
| 802.13 | 基于有线电视的广域通信网 |

*(讲义 p142)*

> Caveat: 802.3～802.5 是最基本的技术。课后习题考过"令牌总线媒体访问控制方法的标准是 IEEE 802.4"。*(讲义 p142, 题目课后习题Q14)*

**IEEE 802 参考模型与 OSI 参考模型对应关系**：*(讲义 p142)*

| OSI 参考模型 | IEEE 802 参考模型 |
|---|---|
| 应用层 | 高层 |
| 表示层 | 高层 |
| 会话层 | 高层 |
| 运输层 | 高层 |
| 网络层 | 高层 |
| 数据链路层 | 逻辑链路控制层 LLC |
| 数据链路层 | 介质访问控制层 MAC |
| 物理层 | 物理层 PLS/MAU |
| 传输介质 | 传输介质 |

*(讲义 p142)*

> Caveat: IEEE 802 将 OSI 数据链路层拆分为 LLC（逻辑链路控制）和 MAC（介质访问控制）两个子层，LLC 通过服务访问点 SAP 与高层交互。*(讲义 p142)*

## 三、以太网介质访问控制

**CSMA/CD** ⭐：载波监听多路访问/冲突检测，是一种争用型的介质访问控制协议。CSMA/CD 协议简单、可靠，广泛使用于局域网，处于网中的各个站都能独立地决定数据帧的发送与接收。*(讲义 p143)*

工作原理：当一个节点要发送数据时，首先监听信道；如果信道空闲就发送数据，并继续监听；如果在数据发送过程中监听到了冲突，则立刻停止数据发送，等待一段随机的时间后，重新开始尝试发送数据。*(讲义 p143)*

控制规程的核心问题：解决在公共通道上以广播方式传送数据中可能出现的问题（主要是数据碰撞问题）。控制过程包含四个处理内容：侦听、发送、检测、冲突处理。*(讲义 p143)*

> Caveat (Ch8-选10): CSMA/CD 可以检测碰撞但不需要 ACK 确认；CSMA/CA 不能检测碰撞（无线环境硬件困难），需 ACK 确认。CSMA/CA 也有随机退避机制。*(题目Ch8-选10)*

> Caveat: 课后习题考过"需要提高最短帧长度"的场景——冲突域最大距离不变，网络传输速率提高时，需提高最短帧长度。*(题目课后习题Q5)*

> Caveat: 快速以太网仍使用 CSMA/CD，采用保持最短帧长不变而将最大电缆长度减少到 100m 的方法，使速率提高到 100Mbit/s。*(题目课后习题Q8)*

**令牌环** ⭐：IBM 公司于 70 年代推出。物理上采用星形拓扑结构，逻辑上仍是环形拓扑结构。*(讲义 p143)*

工作原理：令牌按某一方向沿着环路经过环路中的各个节点单方向传递。握有令牌的站点具有发送数据的权力。环中站点接收帧时，将帧中目的地址与本站地址比较：地址相符则将帧放入接收缓冲器并送回环上；地址不符则将帧沿环路径下传。当节点发送完所有数据或持有令牌到达最大时间时，交出令牌。*(讲义 p143)*

令牌环网缺点：需要维护令牌，一旦失去令牌就无法工作，需选择专门的节点监视和管理令牌。*(讲义 p143)*

令牌环网效率与负载关系：*(讲义 p143)*

| 负载情况 | 效率表现 | 原因 |
|---|---|---|
| 轻负载 | 效率降低 | 等待令牌的时间开销 |
| 重负载 | 各节点公平访问且效率高 | 与总线型拓扑结构相反 |

> Caveat: 课后习题两次考到"令牌环网更适合的环境是负载重"（Q4 和 Q9 重复出题），这是高频考点。规定时间必须传完，否则依旧收回令牌。*(题目课后习题Q4/Q9)*

**以太网组网方式**：最常用的以太网有以下 4 种。*(讲义 p143-144)*

| 标准 | 名称 | 传输介质 | 特点 |
|---|---|---|---|
| 10Base-5 | 粗缆以太网 | 同轴电缆（粗缆） | 每段电缆最大长度 500m；使用特殊收发器连接到电缆上，收发器完成载波监听和冲突检测 |
| 10Base-2 | 细缆以太网 | 同轴电缆（细缆） | 每段电缆最大长度 200m；使用 BNC 连接器形成 T 形连接，无源部件 |
| 10Base-T | 双绞线以太网 | 非屏蔽双绞线 | 技术简单、价格低廉、可靠性高、易于实现、易于管理维护升级；目前还在应用的 10Mbps 局域网技术 |
| 10Base-F | 光纤以太网 | 多模光纤 | 传输距离长、安全可靠、可避免电击；常用于建筑物之间连接，能构建园区主干网 |

*(讲义 p143-144)*

> Caveat: 课后习题考过"使用双绞线的以太网为 10Base-T"（Q19）。例题考过"同轴电缆在无中继情况下最大长度不能超过 500m"（例题3，对应 10Base-5）。*(题目课后习题Q19, 例题3)*

**快速以太网** ⭐：100Mbps 以太网又称快速以太网，传输速率达到 100Mbps。保留了传统以太网的所有特性，只是将每个比特的发送时间由 100ns 降低到 10ns。*(讲义 p144)*

1995 年新的物理层标准(100Base-T)定义了介质专用接口(MII)，将 MAC 子层与物理层分开，使物理层在实现 100Mbps 时所使用的传输介质和信号编码方式的变化不会影响 MAC 子层。*(讲义 p144)*

三种传输介质标准：100Base-TX、100Base-T5、100Base-FX。*(讲义 p144)*

**100Base-TX 组网规则**：*(讲义 p144)*

1. 各网络站点需通过 Hub 连入网络中。
2. 传输介质用 S 类无屏蔽双绞线或 150 Ω 屏蔽双绞线。
3. 使用 8 针 RJ-45 标准连接器实现双绞线与网卡或与 Hub 之间的连接。
4. 各网络站点与 Hub 之间的最大距离为 100m。
5. 一个冲突域中只能连接一个 I 类 Hub，网络最大直径（站点-Hub-站点）为 200m。
6. 使用 II 类 Hub，最多可级连两个 II 类 Hub，网络最大直径（站点-Hub-Hub-站点）为 205m。

*(讲义 p144)*

## 四、无线局域网

**WLAN 概念** ⭐：无线局域网 WLAN（Wireless Local Area Network）是一种利用电磁波传送和接收数据、实现传统有线局域网功能的计算机网络系统。通过无线方式发送和接收数据，减少了对固定线路的依赖，摆脱了有线传输介质的束缚。*(讲义 p145)*

**WLAN 特点** ⭐：*(讲义 p145)*

| 特点 | 说明 |
|---|---|
| 安装便捷 | 免去或减少了网络布线工作量，一般安装一个或多个接入点 AP 设备即可建立覆盖整个建筑或地区的局域网络 |
| 高移动性 | 各结点可随意移动，不受地理位置限制；AP 覆盖范围 10m–100m；在无线信号覆盖范围内终端设备均可接入网络，且能在不同网络间漫游 |
| 易扩展性 | 每个 AP 可支持 100 多个用户接入；扩充网络容量只需增加 AP |
| 经济节约 | 避免或减少有线网络预设大量利用率较低的信息点的不足，节约网络造价，改造时花费也较低 |
| 传输速率高 | 数据传输速率已达 11Mbps，传输距离可远至 20km 以上；使用 OFDM 技术可达 54Mbps |

*(讲义 p145)*

**WLAN 应用**：已在移动办公、不易布线的地方以及远距离的数据处理方面得到应用，特别是在医护管理、会议展览、金融与旅游服务、移动办公等行业中。*(讲义 p145)*

**WLAN 协议（802.11 系列）** ⭐：*(讲义 p145)*

| 标准 | 数据传输速率 |
|---|---|
| 802.11n 草案 | 200+ Mbps |
| 802.11a/g | 54 Mbps |
| 802.11b | 5.5 – 11 Mbps |
| 802.11（原始标准） | 1 – 2 Mbps |

*(讲义 p145)*

覆盖范围：802.11 的覆盖范围约为 30m 和 60m。*(讲义 p145)*

> Caveat (Ch8-选10): CSMA/CA 是无线局域网标准 802.11 中的协议，利用 ACK 信号来避免冲突发生——只有当客户端收到网络上返回的 ACK 信号后才确认送出的数据已经正确到达目的地址。CSMA/CD 接收方不需要确认。*(题目Ch8-选10, 例题4)*

## 五、广域网技术

**ISDN**：综合业务数字网（Integrated Service Digital Network），利用公众电话网向用户提供端对端的数字信道连接，承载包括话音和非话音在内的各种电信业务。普遍开放的是 N-ISDN（窄带 ISDN）。俗称"一线通"。*(讲义 p146)*

两种速率接入方式：*(讲义 p146)*

| 接入方式 | 信道结构 | B 信道速率 | D 信道速率 | 联网设备 |
|---|---|---|---|---|
| BRI（基本速率接口） | 2B+D | 64Kbps | 16Kbps | NT1 或 NT1 Plus |
| PRI（基群速率接口） | 30B+D | 64Kbps | 64Kbps | HDSL（市话双绞线）或光 Modem 及光端机（光纤） |

*(讲义 p146)*

B 信道传输话音、数据和图像；D 信道传输信令或分组信息（现尚未开放业务）。*(讲义 p146)*

ISDN 相比 DDN 和帧中继的优势：*(讲义 p146)*

1. 业务实现方便，提供的业务种类丰富——基于现有公众电话网，电话交换机有 ISDN 功能模块即可提供业务。
2. 用户使用灵活便捷——2B+D 可作为两部电话同时使用，也可 64Kbps 联网，还可 128Kbps 速率联网；30B+D 可灵活高速联网。
3. 适宜的性价比——按 B 信道通信计费，1B 信道国内通信费率等同于普通电话费率，通信量较少的用户费用远低于租用 DDN 专线或帧中继。

*(讲义 p146)*

**XDSL 与 ADSL** ⭐：XDSL 是 HDSL、ADSL、VDSL 等技术的统称。HDSL 主要支持 2Mbps 及以下速率；VDSL 速率可达 25Mbps 以上但线路长度较短（25Mbps 时约 1Km）。ADSL（Asymmetric Digital Subscriber Line）在实际使用中最为普遍。*(讲义 p146-147)*

ADSL 优势：*(讲义 p147)*

1. 在一对双绞线上为用户提供高达 8Mbps 的下行速率和 1Mbps 的上行速率。下行速率大于上行速率，符合普通用户联网实际需要。
2. 不影响用户对普通电话的使用——使用独特的信号调制技术，接入 ADSL 的同时仍可进行普通电话通信。
3. 调制技术：DMT（离散多频音调制）和 CAP（无载波调幅/调相）两种。DMT 已被 ANSI 采纳为标准，CAP 技术还在进一步完善。

*(讲义 p147)*

ADSL 传输距离限制：信号在从局端发送到用户端的途中会迅速衰减，有质量保证的一般在 3 公里以内（如 0.4mm 线径双绞线，速率为 3Mbps 时传输距离约 2.7Km）。对于 3～5 公里的用户可采用 FTTC+ADSL 方式（在 ONU 上集成 ADSL 用户接口板）。*(讲义 p147)*

ADSL 组网关键组件：用户侧——个人计算机 → ATU-R（ADSL Modem）→ 电话外线；电话公司侧——ATU-C（ADSL Modem）→ 后端滤波器 → 程控交换机 → Internet。普通电话用户通过电话外线直接接入，不受 ADSL 影响。*(讲义 p147)*

**HFC**：Hybrid Fiber/Coax，网络传输主干为光纤，到用户端为同轴电缆的用户网络接入方式。我国各城市有线电视网按电信网络要求升级改造后可提供 HFC 接入，实现普通电话、VOD、远程医疗等窄带和宽带业务。*(讲义 p147-148)*

HFC 优势：*(讲义 p147-148)*

1. 信号通频带宽为 750 MHz，为市话双绞线无法比拟。
2. 能充分适应信息网络发展，易于过渡为最终的 FTTH/O 方式——光节点所带用户数可逐渐减少直至仅带一个用户。
3. 成本费用很低，提供宽带接入的费用明显低于 XDSL 等接入费用。

HFC 劣势：有线电视网带宽为所有用户共享，每一用户所占带宽不固定，取决于某一时刻共享用户数。用户增加时每个用户分得的实际带宽将明显降低，甚至低于用户独享的 ADSL 带宽。*(讲义 p148)*

## 六、重点对比与易混辨析

### CSMA/CD vs CSMA/CA

| 对比项 | CSMA/CD | CSMA/CA |
|---|---|---|
| 全称 | 载波监听多路访问/冲突检测 | 载波监听多路访问/冲突避免 |
| 检测碰撞 | 可以检测碰撞 | 不能检测碰撞（无线环境硬件困难） |
| ACK 确认 | 不需要 ACK | 需要 ACK 确认 |
| 应用场景 | 有线局域网（802.3） | 无线局域网（802.11） |
| 随机退避 | 有 | 有 |
| 标准归属 | IEEE 802.3 | IEEE 802.11 |

*(讲义 p143, p145; 题目Ch8-选10, 例题4)*

### CSMA/CD vs 令牌环

| 对比项 | CSMA/CD | 令牌环 |
|---|---|---|
| 介质访问方式 | 争用型 | 令牌传递 |
| 负载特性 | 轻负载效率高，重负载冲突增多 | 轻负载效率低（等令牌），重负载公平且效率高 |
| 维护复杂度 | 低 | 需维护令牌，失去令牌无法工作 |
| 拓扑结构 | 总线型/星型 | 物理星型，逻辑环型 |
| 适用环境 | 一般负载 | 重负载环境 |

*(讲义 p143)*

### ADSL vs HFC

| 对比项 | ADSL | HFC |
|---|---|---|
| 传输介质 | 双绞线 | 光纤主干 + 同轴电缆到用户 |
| 带宽方式 | 用户独享 | 所有用户共享 |
| 下行速率 | 高达 8Mbps | 取决于共享用户数 |
| 上行速率 | 1Mbps | [原文未述] |
| 通频带宽 | [原文未述] | 750 MHz |
| 成本 | 较高 | 很低 |
| 传输距离限制 | 有质量保证一般在 3 公里以内 | [原文未述] |
| 演进方向 | FTTC+ADSL | 可过渡为 FTTH/O |

*(讲义 p147-148)*

### ISDN BRI vs PRI

| 对比项 | BRI | PRI |
|---|---|---|
| 信道结构 | 2B+D | 30B+D |
| B 信道速率 | 64Kbps | 64Kbps |
| D 信道速率 | 16Kbps | 64Kbps |
| 联网设备 | NT1 或 NT1 Plus | HDSL（双绞线）或光 Modem 及光端机（光纤） |

## 七、高频考点与易错提醒

1. **CSMA/CD 与 CSMA/CA 区别** ⭐⚠：CSMA/CD 可检测碰撞、不需 ACK；CSMA/CA 不能检测碰撞、需 ACK 确认。CSMA/CA 用于无线局域网 802.11，CSMA/CD 用于有线以太网 802.3。*(题目Ch8-选10, 例题4)*

2. **令牌环网负载特性** ⭐⚠：令牌环网在重负载时效率高、公平访问；轻负载时因等待令牌效率降低。考题两次重复考查"令牌环网更适合负载重环境"。*(题目课后习题Q4/Q9)*

3. **CSMA/CD 最短帧长** ⭐：冲突域最大距离不变、网络传输速率提高时，需要提高最短帧长度。快速以太网保持最短帧长不变，通过减少最大电缆长度到 100m 来提高速率。*(题目课后习题Q5/Q8)*

4. **IEEE 802 标准对应** ⭐：802.3 → CSMA/CD，802.4 → 令牌总线，802.5 → 令牌环，802.11 → WLAN。802.3～802.5 是最基本的技术。*(讲义 p142, 题目课后习题Q14)*

5. **以太网物理地址长度** ⭐：IEEE 802.3 标准以太网物理地址长度为 48 位。*(例题1)*

6. **MAC 地址易错点** ⚠：MAC 地址又称为局域网硬件地址，通常存储在网卡中，不是通过域名解析查得的。同一局域网两个设备具有相同静态 MAC 地址时，两个设备都不能正确通信。*(题目课后习题Q6/Q10)*

7. **以太网组网方式识别** ⭐：10Base-5（粗缆，500m），10Base-2（细缆，200m），10Base-T（双绞线），10Base-F（光纤）。*(讲义 p143-144, 题目课后习题Q19)*

8. **局域网访问冲突根源** ⭐：局域网中访问冲突的根源是共享介质。*(题目课后习题Q18)*

9. **坚持 CSMA 含义** ⭐：坚持 CSMA 方式中"坚持"的含义是——当侦听到总线忙时，坚持侦听直到总线空闲为止。*(题目课后习题Q17)*

10. **OSI 设备层次映射** ⚠：广域网交换机工作在数据链路层；路由器是网络层设备；中继器和集线器是物理层设备。*(例题2)*

11. **HFC 带宽共享特性** ⚠：HFC 带宽为所有用户共享，用户增加时每用户实际带宽降低；ADSL 带宽为用户独享。*(讲义 p148)*

## 八、一句话记忆要点

- **局域网**：覆盖小、速率高、协议简单，只涉及物理层和数据链路层。
- **IEEE 802**：LLC + MAC 拆分链路层；802.3 以太网、802.4 令牌总线、802.5 令牌环、802.11 无线局域网。
- **CSMA/CD**：先听后发、边发边听、冲突停发、随机重发；有线用、能检测碰撞、不需 ACK。
- **CSMA/CA**：无线用、不能检测碰撞、需 ACK 确认、有随机退避。
- **令牌环**：物理星型逻辑环型，轻负载低效、重负载高效公平。
- **快速以太网**：100Mbps，比特发送时间 100ns→10ns，保持最短帧长不变、电缆减到 100m。
- **WLAN**：AP 覆盖 10–100m，802.11n 200+Mbps、802.11a/g 54Mbps、802.11b 5.5–11Mbps。
- **ISDN**：一线通，BRI=2B+D（D=16Kbps），PRI=30B+D（D=64Kbps）。
- **ADSL**：非对称，下行 8Mbps/上行 1Mbps，DMT 和 CAP 调制，3 公内有质量保证。
- **HFC**：光纤主干+同轴到户，750MHz 带宽共享，成本低但带宽随用户增加而降低。
