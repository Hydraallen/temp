第七章 习题答案及解析

一、单选题：

1. C

解析：系统可用性是指系统中的资源随时都能供授权用户访问。

考点：安全环境的主要目标

2. C

解析：软件被删除后相应的软件处于不可用状态，影响了系统的可用性。

考点：系统可用性概念

3. A

解析：软件未经授权被使用，是软件被泄密了，保密性是指系统仅允许被授权用户访问计算机系统中的信息。

考点：安全环境的主要目标

4. B

解析：完整性是指系统中所保存的信息既不会丢失，也不会被非授权用户修改，且能保持数据的一致性。

考点：完整性的概念

5. C

解析：数据加密技术包括数据加密和解密、数字签名、签名识别和数字证明书等几个方面的内容，它是保障计算机系统和网络安全的最基本、最主要的技术。

考点：数据加密技术

6. A

解析：数据截取是指未经核准的用户通过非正当的途径(如直接从电话线上窃听)截取网络中的文件和数据。数据机密性是指系统仅允许被授权的用户访问计算机系统中的信息。

考点：数据机密性的概念

7. C

解析：最基本的加密算法是易位法和置换法。

考点：加密方法

8. C

解析：DES 是最具代表性的对称加密算法

考点：DES 算法

9. C
