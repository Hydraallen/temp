## 二、知识点详解

### 1. 局域网的概念 ⭐

- **定义**：一般而言，在小范围内将多种通信设备互连起来，实现数据通信和资源共享的计算机网络。
- **主要特点**：
  1. 覆盖范围小。
  2. 只能够提供物理层、数据链路层各网络层的通信功能。
  3. 可以连入多种数据通信设备。
  4. 局域网的数据传输速率高，误码率较低可靠性高。
  5. 协议简单、结构灵活、建网成本低、周期短、便于管理和扩充。

> 根据网络覆盖地理范围的大小，计算机网络可分为局域网、城域网和广域网 3 种类型。(练习1，答案 A)

---

### 2. 局域网的拓扑结构

- 局域网常采用**广播型**的拓扑结构，常见的拓扑结构有**总线型、星型和环型**拓扑结构 3 种。

---

### 3. 局域网的分类方式

#### 3.1 按照介质访问控制方法

| 类型 | 介质访问控制方法 |
|------|------------------|
| 以太网 | CSMA/CD 介质访问控制方法 |
| 令牌环网 | 通过"令牌"，让各个结点获得数据发送权，避免数据传输的冲突 |

#### 3.2 按照网络资源管理方式

| 类型 | 特点 |
|------|------|
| **对等式**局域网 | 所有结点地位平等，任何结点之间都可以直接通信和资源共享，各个结点都拥有自主权 |
| **非对等式**网络 | 结点的地位有所不同，如服务器与客户机。服务器通过集中控制的方式管理网络资源，并为工作站提供服务。"客户机／服务器"网络是非对等式网络的典型代表。管理分散，任何人都可以成为管理者，需要授权；资源相对集中；多见办公室、家庭 |

#### 3.3 按照网络传输技术

| 类型 | 传输技术 |
|------|----------|
| 基带局域网 | 采用数字信号基带传输技术 |
| 宽带局域网 | 采用模拟信号频带信号传输技术 |

### 5. CSMA/CD ⭐⚠

- **全称**：载波监听多路访问 / 冲突检测
- **性质**：一种**争用型**的介质访问控制协议。CSMA/CD 协议简单、可靠，广泛使用于局域网，处于网中的各个站都能独立地决定数据帧的发送与接收。

#### 工作原理

```
当一个节点要发送数据时：
1. 首先监听信道；
2. 如果信道空闲就发送数据，并继续监听；
3. 如果在数据发送过程中监听到了冲突，
   则立刻停止数据发送，
   等待一段随机的时间后，重新开始尝试发送数据。
```

#### CSMA/CD 控制规程

- **核心问题**：解决在公共通道上以广播方式传送数据中可能出现的问题（主要是**数据碰撞问题**）。
- **控制过程包含四个处理内容**：侦听、发送、检测、冲突处理。

> ⚠ 根据练习5（答案 B），冲突域的最大距离不变、网络传输速率提高时，需要提高最短帧长度。(练习5)
>
> ⚠ 快速以太网仍使用 CSMA/CD，采用**保持最短帧长不变**而将最大电缆长度减少到 100m 的方法，使速率提高到 100Mbit/s。(练习8，答案 C)
>
> ⚠ 坚持CSMA方式中"坚持"的含义是：当侦听到总线忙时，坚持侦听直到总线空闲为止。(练习17，答案 A)
>
> ⚠ 局域网中访问冲突的根源是**共享介质**。(练习18，答案 B)

### 6. 令牌环 ⭐

- **推出者**：IBM 公司于 70 年代推出。
- **拓扑结构**：物理上采用了**星形**拓扑结构，逻辑上仍是**环形**拓扑结构。

#### 工作原理

```
1. 在网络正常工作时，令牌按某一方向沿着环路经过环路中的各个节点单方向传递。
2. 握有令牌的站点就具有了发送数据的权力。
3. 在环中的站点接收发送的帧时，将帧中的目的地址与本站的地址进行比较：
   - 如地址相符：则将帧放入接收缓冲器，并将帧送回环上；
   - 如地址不符：则将帧沿环的路径下传。
4. 当该节点发送完所有数据或者持有令牌到达最大时间时，就要交出令牌。
```

#### 缺点与负载特性

- 需要维护令牌，一旦失去令牌就无法工作，需要选择专门的节点监视和管理令牌。
- 效率高低由**负载高低**决定：
  - **轻负载**时：由于在等待令牌的时间，效率降低了；
  - **重负载**时：对各节点公平访问且效率高（与总线型拓扑结构**相反**）。
- 规定时间必须传完，否则依旧收回令牌。

> ⭐ 令牌环网更适合**负载重**的环境。(练习4/9，答案 B)

### 7. 以太网组网方式 ⭐

最常用的以太网有以下 4 种：

| 标准 | 名称 | 传输介质 | 特点 |
|------|------|----------|------|
| **10Base-5** | 粗缆以太网 | 同轴电缆（粗） | — |
| **10Base-2** | 细缆以太网 | 同轴电缆（细） | — |
| **10Base-T** | 双绞线以太网 | 非屏蔽双绞线 | 技术简单、价格低廉、可靠性高、易于实现，易于管理、维护、升级。目前还在应用的 10Mbps 局域网技术 |
| **10Base-F** | 光纤以太网 | 多模光纤 | 传输距离长、安全可靠、可避免电击。常用于建筑物之间的连接，能够构建园区主干网 |

> ⭐ 使用双绞线的以太网为 10Base-T。(练习19，答案 C)
>
> ⚠ 例题3指出：10Base-5（粗缆）每段电缆最大长度 500m；10Base-2（细缆）每段电缆最大长度 200m。(例题3)

### 8. 快速以太网（100Mbps 以太网）⚠

- **又称**：快速以太网，传输速率达到 **100Mbps**。
- **核心特性**：保留了传统以太网的所有特性，只是将每个比特的发送时间由 **100ns** 降低到 **10ns**。
- **物理层标准**：1995 年新的物理层标准（**100Base-T**）定义了**介质专用接口（MII）**，它将 MAC 子层与物理层分开，使物理层在实现 100Mbps 时所使用的传输介质和信号编码方式的变化不会影响 MAC 子层。
- **三种传输介质标准**：100Base-TX、100Base-T5[?]、100Base-FX。

#### 100Base-TX 组网规则

1. 各网络站点需通过 Hub 连入网络中。
2. 传输介质用 S 类[?]无屏蔽双绞线或 150 Ω[原文为 Q] 屏蔽双绞线。
3. 使用 8 针 RJ-45 标准连接器实现双绞线与网卡，或与 Hub 之间的连接。
4. 各个网络站点与 Hub 之间的最大距离为 **100m**。
5. 在一个冲突域中只能连接一个 I 类 Hub，网络的最大直径（站点-Hub-站点）为 **200m**。
6. 如果使用 II 类 Hub，最多可以级连两个 II 类 Hub，网络的最大直径（站点-Hub-Hub-站点）为 **205m**。

> ⚠ 快速以太网保持最短帧长不变，通过减小最大电缆长度来提高速率。(练习8，答案 C)

### 9. 无线局域网基本概念与特点

#### 9.1 基本概念

- **WLAN（Wireless Local Area Network）**：是一种利用电磁波传送和接收数据、实现传统有线局域网的功能的计算机网络系统。
- 无线局域网是一种灵活的数据通信系统，通过无线方式发送和接收数据，减少了对固定线路的依赖，摆脱了有线传输介质的束缚。

#### 9.2 基本特点

| 特点 | 说明 |
|------|------|
| **安装便捷** | 免去或减少了网络布线的工作量，一般只要安装一个或多个接入点 AP（Access Point）设备，就可建立覆盖整个建筑或地区的局域网络 |
| **高移动性** | 各结点可随意移动，不受地理位置的限制。AP 的覆盖范围是 **10m—100m**，能够在不同的网络间漫游 |
| **易扩展性** | 每个 AP 可支持 **100多个**用户的接入。需要扩充网络容量，只需增加 AP 即可 |
| **经济节约** | 避免或减少有线网络预设大量利用率较低的信息点的不足，节约网络造价，改造时花费也较低 |
| **传输速率高** | 数据传输速率现在已达 **11Mbps**，传输距离可远至 **20km** 以上。使用正交频分复用（OFDM）技术，传输速率可以达到 **54Mbps** |

#### 9.3 应用

- 移动办公、不易布线的地方以及远距离的数据处理方面，特别是在医护管理、会议展览、金融与旅游服务、移动办公等行业中。

### 11. ISDN

- **全称**：Integrated Service Digital Network，综合业务数字网。
- **功能**：利用公众电话网向用户提供了端对端的数字信道连接，用来承载包括话音和非话音在内的各种电信业务。
- **现状**：现在普遍开放的 ISDN 业务为 **N-ISDN**（窄带 ISDN）。
- **俗称**："一线通"。

#### 两种速率接入方式

| 接入方式 | 信道结构 | 信道速率 | 联网设备 |
|----------|----------|----------|----------|
| **BRI**（Basic Rate Interface） | 2B+D | B 信道 = 64Kbps（2个），D 信道 = 16Kbps（1个） | NT1 或 NT1 Plus 设备 |
| **PRI**（Primary Rate Interface） | 30B+D | B 和 D 皆为 64Kbps | HDSL 设备（市话双绞线）或光 Modem 及光端机（光纤） |

- **B 信道**一般用来传输话音、数据和图像。
- **D 信道**用来传输信令或分组信息（现尚未开放业务）。

#### ISDN 的优势（与 DDN 和帧中继相比）

1. **业务实现方便，提供的业务种类丰富**：ISDN 基于现有的公众电话网，凡是普通电话覆盖到的地方，只要电话交换机有 ISDN 功能模块，即可为用户提供 ISDN 业务。业务种类繁多，包括普通电话、联网、可视电话等基本业务及主叫号码显示等许多补充业务。
2. **用户使用非常灵活便捷**：对于 2B+D，用户既可作为两部电话同时使用，又可以 64Kbps 联网，另一 64Kbps 用于普通电话；还可根据需要以 **128Kbps** 速率联网。而 30B+D 可使用户灵活、高速联网。
3. **适宜的性价比**：ISDN 按使用的 B 信道进行通信计费，1B 信道的国内通信费率等同于普通电话通信费率。对于通信量较少、通信时间较短的用户，费用远低于租用 DDN 专线或帧中继电路的费用。

### 12. ADSL

#### 12.1 XDSL 概述

- **XDSL** 是 HDSL、ADSL、VDSL 等技术的统称。
  - **HDSL**：主要支持 2Mbps 及以下的速率。
  - **VDSL**：速率可达 25Mbps 以上，但线路长度较短（25Mbps 时约为 1Km），且部分技术尚未完全确定。
  - **ADSL**（Asymmetric Digital Subscriber Line）：在实际使用中最为普遍。

#### 12.2 ADSL 技术概述

- ADSL 的技术标准出台于 1997 年。在 HDSL 技术的基础上，根据网络和用户间的业务流量特点，在信号调制、数字相位均衡、回波抵消等方面采用了更为先进的器件和动态控制技术。

#### 12.3 ADSL 的优势

1. **高速率**：在一对双绞线上可为用户提供高达 **8Mbps** 的下行速率，**1Mbps** 的上行速率。下行速率大于上行速率，非常符合普通用户联网的实际需要。较充足的带宽可用于传输多种宽带数据业务，如会议电视、VOD、HDTV 业务等。
2. **不影响普通电话使用**：由于使用了独特的信号调制技术，用户接入 ADSL 的同时仍然可以进行普通电话通信。
3. **调制技术**：目前应用最为广泛的为 **DMT（离散多频音调制）** 和 **CAP（无载波调幅/调相）** 两种。其中，DMT 已经被 ANSI 采纳为标准，而 CAP 技术还正在进一步完善。

#### 12.4 传输距离限制

- ADSL 业务的信号在从局端发送到用户端的途中会迅速衰减，有质量保证的一般在 **3 公里**以内（如对 0.4mm 线径的双绞线，速率为 3Mbps 时的传输距离约为 2.7Km）。
- 目前市话端局的覆盖面积一般在 5 公里以内，对于 3～5 公里的用户必须考虑采用其他方法。
- 在市区内，随着 **FTTC（光纤到路边）** 覆盖范围的扩展，可在 ONU 上集成 ADSL 的用户接口板，即 **FTTC+ADSL** 方式。

### 13. HFC

- **全称**：Hybrid Fiber/Coax，即网络传输主干为光纤，到用户端为同轴电缆的用户网络接入方式。
- 我国各城市的有线电视网按照电信网络的要求进行一定的升级改造，即可为用户提供 HFC 接入，实现普通电话、VOD、远程医疗等窄带和宽带业务。

#### 优势

1. **通频带宽为 750 MHz**，为市话双绞线所无法比拟。
2. 能充分适应信息网络的发展，易于过渡为最终的 **FTTH/O** 方式。如现有有线电视网络的一个光节点（相当于 ONU）所带的用户数可由现在的数百户逐渐减少，直至最终仅带一个用户，此时 HFC 方式即过渡为最终的 FTTH/O 方式。
3. 同利用 ADSL 等电信网络实现宽带接入的成本相比，它的成本费用很低。
4. 由它提供宽带接入的费用会明显低于 XDSL 等的接入费用。这是它在应用中最大的优势。

#### 劣势

- 有线电视网的带宽为**所有用户所共享**。即每一用户所占的带宽并不固定，它取决于某一时刻对带宽进行共享的用户数。随着用户的增加，每个用户分得的实际带宽将明显降低，甚至低于用户独享的 ADSL 带宽。

## 三、公式 / 算法 / 定理速查表

### CSMA/CD 工作流程

```
1. 监听信道 → 空闲则发送数据并继续监听
2. 发送过程中检测到冲突 → 立刻停止发送
3. 等待一段随机时间 → 重新尝试发送
```

### 令牌环工作流程

```
1. 令牌沿环路单方向传递
2. 握有令牌的站点获得发送权
3. 目的地址相符 → 接收；不符 → 转发
4. 发送完毕或持令牌达最大时间 → 交出令牌
```

## 五、高频考点与易错提醒

> 以下考点信号来自本章例题、课后习题及书末统一答案，仅列考点方向与可确认结论。

1. ⭐ **网络按覆盖范围分为局域网、城域网、广域网**，划分依据是网络的作用范围。(练习1，答案 A)
2. ⭐ **局域网和广域网的差异**不仅在于覆盖范围，还主要在于所使用的协议不同。(练习2，答案 B)
3. ⭐ **因特网采用的核心技术是 TCP/IP**。(练习3，答案 A)
4. ⭐ **令牌环网更适合负载重的环境**，CSMA/CD 更适合负载轻的环境。(练习4/9，答案 B)
5. ⚠ **CSMA/CD 最短帧长**：冲突域最大距离不变、网络传输速率提高时，需要提高最短帧长度。(练习5，答案 B)
6. ⚠ **相同静态 MAC 地址**：在同一局域网的两个设备具有相同的静态 MAC 地址时，这两个设备都不能正确通信。(练习6，答案 C)
7. ⚠ **以太网广播**：大量广播信息降低网络性能的原因是网络中的每台计算机都必须处理每个广播信息。(练习7，答案 B)
8. ⚠ **快速以太网**：仍使用 CSMA/CD，采用**保持最短帧长不变**而将最大电缆长度减少到 100m 的方法提高速率到 100Mbit/s。(练习8，答案 C)
9. ⚠ **MAC 地址**：MAC 地址不是通过域名解析查得的，以太网地址通常存储在网卡中。(练习10，答案 C)
10. ⭐ **Internet 网络层四个重要协议**：IP、ICMP、ARP、RARP。(练习11，答案 C)
11. ⚠ **TCP 协议**通过 IP 地址+端口号来区分不同的连接。(练习13，答案 C)
12. ⭐ **令牌总线媒体访问控制方法的标准是 IEEE 802.4**。(练习14，答案 B)
13. ⭐ **T1 速率 = 1.544Mbps**（北美 24 路 PCM）。(练习15，书末答案解析 C)
14. ⭐ **E1 速率 = 2.048Mbps**（欧洲 30 路 PCM）。(练习16，书末答案解析 D)
15. ⚠ **坚持 CSMA** 中"坚持"的含义是：当侦听到总线忙时，坚持侦听直到总线空闲为止。(练习17，答案 A)
16. ⚠ **局域网中访问冲突的根源是共享介质**。(练习18，答案 B)
17. ⭐ **使用双绞线的以太网为 10Base-T**。(练习19，答案 C)
18. ⚠ **B 类地址**判断：首字节范围 128–191，156.123.32.12 属于 B 类。(练习20，答案 D)
19. ⚠ **IEEE 802.3 物理地址长度为 48bit**。(例题1，答案 C)

## 六、一句话记忆要点

1. **局域网**：覆盖小、速率高、协议简单、只涉及物理层和数据链路层。
2. **IEEE 802**：LLC + MAC 两子层替代 OSI 数据链路层；802.3=CSMA/CD，802.4=令牌总线，802.5=令牌环，802.11=WLAN。
3. **CSMA/CD**：先听后发、边发边听、冲突停发、随机重发——争用型，适合轻负载。
4. **令牌环**：令牌单向传递，握有令牌才能发送——重负载下公平且高效，轻负载效率低。
5. **以太网组网**：5=粗缆、2=细缆、T=双绞线、F=光纤；快速以太网保持最短帧长不变、缩短电缆到 100m。
6. **WLAN**：AP 覆盖 10–100m，每 AP 支持 100+ 用户，OFDM 可达 54Mbps。
7. **ISDN**："一线通"，BRI=2B+D（64K×2+16K），PRI=30B+D（64K×31）。
8. **ADSL**：下行 8Mbps / 上行 1Mbps，不影响电话，DMT 为 ANSI 标准，3 公内有质量保证。
9. **HFC**：光纤主干+同轴到户，带宽 750MHz，带宽共享是最大劣势，成本低于 ADSL 是最大优势。
10. **T1/E1**：T1=1.544Mbps（北美24路），E1=2.048Mbps（欧洲30路）。
