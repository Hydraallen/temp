# 水冷（CPU 一体式 AIO）选购

> 所有型号、规格、价格均来自真实联网搜索，文末附来源。凡公开资料未明示、需装前核对的，标"待核实"。
> **散热对象是 AMD Ryzen 9 9950X**（16C32T，TDP 170W，PPT/满载实测约 200–230W），是出了名的高发热、难压 U。
> **机箱是华硕 TUF GT502 Horizon ARGB**，冷排只能顶置、最高 360mm——所以**只选 360mm AIO**，"原生 AM5 扣具 + 能顶置 360"是硬否决项。

## 一、需求与硬指标

**配置**：R9 9950X（AM5 / LGA1718，满载 200W+）+ ROG Crosshair X870E DARK HERO（ATX）+ 华硕 TUF RTX 5090 风冷版（348mm / 3.6 槽，走 12V-2×6）+ 华硕 TUF GT502 Horizon ARGB（顶置最高 360mm，后置仅 120mm，出厂 7 把 ARGB 风扇含顶部 3 把）。

**硬指标（须同时满足）**：
- **A** 原生支持 **AM5（LGA1718）** 扣具——硬否决项
- **B** 冷排规格 = **360mm**（顶部 3×120；若顶部支持 140，280 亦可，但优先 360）——硬否决项
- **C** 能扛住 **9950X 满载 200W+** 不长时间撞 95℃ 温度墙（越厚排/越强泵越有余量）
- **D** 冷头高度、水管出线方向不与 **RTX 5090 的 12V-2×6 供电线**打架（顶置走线要避让）
- **E** 顶置会占用出厂 3 把顶部 ARGB 扇位——冷排自带扇即可顶替，风道分工：顶排风、侧进风（GT502 侧面反叶扇正对显卡）

**关于"9950X 能不能压住"的关键认知（重要）**：
9950X 是**热密度**受限，不是单纯冷排面积受限。它的发热集中在很小的 CCD die 上，热量"传不出来"的瓶颈在硅片本身。**即便是全场最强 360/420 AIO，满载 PBO 全核也普遍在 80–95℃**，这是正常的、不是水冷不行。所以选水冷的目标不是"压到很凉"，而是**留足余量、避免长时间贴着 95℃ 撞温度墙降频**。结论：**别买低端薄排/弱泵**，但也别指望任何 360 能把它压到 60 多度。

## 三、xlsx 两个型号点评

### 1. 利民 AE360 V3（Aqua Elite 360 V3）——豆包方案选的，约 ¥280
- **它是什么**：真·360mm 一体水冷，三相九极马达泵 + 3 把 ARGB PWM 扇（500–2000rpm），原生支持 AM5 + Intel LGA1700。利民 360 水冷里最便宜的走量款。
- **优点**：便宜（约 280–293），规格达标（360 排 + 原生 AM5 + 顶置无压力），ARGB 好看，装 GT502 完全兼容。
- **缺点/够不够**：它是**入门级 27mm 薄排 + 走量泵**。压 9950X **能用但余量最小**——满载会偏热、更容易贴近温度墙，日常/游戏没问题，长时间全核渲染/AVX 重载不如厚排（Arctic）或强泵（LT720）从容。
- **结论**：预算极限、以日常+游戏为主，可以上；但既然 U 是 9950X 这种高发热 U，**更推荐加一两百上 Frozen Warframe / LT720**，换更大余量和更稳的降频表现。

### 2. 超频三 臻 RZ620——方案一选的，约 ¥299（**注意：这是风冷，不是水冷！**）
- **核实结论**：**RZ620 是双塔空气散热器**（6 热管、塔高 157.5mm、金属阳极顶盖、支持 AM5/1700），**不是一体水冷**。搜索与规格页均确认为风冷。
- **能压住 9950X 吗**：网友讨论中它能压住 9900X 级别，但 **9950X 满载 200W+ 明显更难**。双塔风冷压 9950X 满载普遍很吃力、容易撞温度墙，尤其在 GT502 这种顶置排风、机箱内还有 575W 5090 灌热的环境里，进气温度更高、风冷更不利。
- **xlsx 备注"建议用 360 水冷"的含义**：就是**在承认 RZ620 风冷压不太住 9950X**——所以配置表才注明"建议改用 360 水冷"。换句话说，RZ620 这一行**本质是被否掉的方案**，最终应落到 360 AIO。
- **结论**：**不要把 RZ620 当水冷买**。若坚持风冷，9950X 请上顶级双塔（如利民 FROZEN NOTTE / Peerless Assassin 之上的旗舰、猫头鹰 NH-D15 G2 级别），但在本机箱 + 高发热 U 场景下，**360 水冷仍是更稳的选择**。

## 四、推荐结论（3 档）

- **🥇 性价比档：利民 Frozen Warframe 360 ARGB（约 ¥417）** 或 **利民 AE360 V3（约 ¥280）**
  - 预算最紧、日常+游戏为主 → AE360 V3 即可达标。
  - 只多花一百多 → **Frozen Warframe 360** 泵更强（3000rpm）、做工更好，对 9950X 余量更足，是这一档更稳的选择。两者都原生 AM5、27mm 排、GT502 顶置零压力。

- **🥈 均衡档（最推荐日常首选）：九州风神 DeepCool LT720（约 ¥489–599）**
  - 第 4 代高流速泵，高端 360 一线水平，压 9950X 余量足；27mm 标准排，GT502 顶置无兼容风险；无带屏溢价，价格甜。**综合最省心。**

- **🥉 高端 / 极限压制：Arctic Liquid Freezer III (Pro) 360**（若国行渠道可得）；**带屏党：NZXT Kraken Elite 360（约 ¥1999）/ Corsair iCUE Link H150i LCD / 利民 Frozen Warframe Ultra（带屏约 ¥599）**
  - **要最强压制**：Arctic Liquid Freezer III Pro 360（38mm 厚排 + 泵头 VRM 辅助扇），Tom's Hardware 评为当前最佳 AIO，对 9950X 余量最大——**唯一注意点**是 38mm 厚排 + 扇约 63mm，GT502 顶置**限高需装前核实**（见五）；且国行/京东价与货源**待核实**。
  - **要带屏颜值**：Kraken Elite 360（640×640 高清屏）最成熟；预算有限想要屏 → 利民 Frozen Warframe **Ultra**（2.88″ 屏，约 599）性价比高。

**一句话首选**：追求最省心且余量足 → **九州风神 LT720**；追求极限压制且能确认厚排装得下 → **Arctic Liquid Freezer III Pro 360**；预算优先 → **利民 Frozen Warframe 360 / AE360 V3**。

## 五、避坑提醒

1. **别买错——风冷 vs 水冷**：xlsx 里的**超频三 RZ620 是风冷双塔**，不是水冷；配置表"建议用 360 水冷"就是让你**别用它压 9950X**。要顶置水冷方案就直接买 360 AIO。
2. **AM5 扣具**：本文候选**全部原生支持 AM5（LGA1718）**，无需额外买扣具；但京东下单前仍核对详情页"支持平台"含 AM5/AM4，避免买到只标 Intel 的旧批次。
3. **厚排的顶置限高（Arctic / T30 这类 38mm 厚排）**：38mm 排 + 25mm 扇 ≈ 63mm。GT502 官方只标"顶部支持 360mm"，**未明示顶置冷排厚度上限**——厚排 + 顶部风扇 + 主板供电/内存马甲高度叠加后是否顶到，**装前务必核实**（量一下主板顶沿到机箱顶盖内侧的净空）。普通 **27mm 排（LT720 / Frozen Warframe / Kraken / H150i / AE360）无此顾虑**。
4. **水管方向 / 冷头出线避让 12V-2×6**：顶置时让**水管走远离显卡供电侧的一端**（通常靠机箱后方或 I/O 侧），冷头软管别压向显卡 12V-2×6 插头方向；5090 的供电线要求插头 35mm 内不弯折，水管/冷排别顶到它。
5. **顶置进/排风方向**：顶排建议**排风**（把 CPU 热量往外抽），配合 GT502 侧面反叶扇给显卡进风、后置 120 排风，形成"侧进—顶后排"的干净风道；别把顶排装成进风，否则把 575W 显卡的热风倒灌进冷排。
6. **顶置会占掉出厂顶部 3 把 ARGB 扇**：GT502 Horizon ARGB 出厂 7 扇（顶 3 + 侧 3 + 后 1），装顶置 360 后**顶部 3 把被冷排自带扇顶替**，拆下的 3 把可挪作它用或闲置；侧面 3 把反叶进风扇保留（对 5090 最关键）。
7. **别为"带屏"过度加价**：屏只是颜值/监控，对 9950X 压制无实质帮助。压制余量优先看**排厚 + 泵**，其次才是屏。
